# 2048

2048 es una versión sencilla y funcional del clásico juego de fichas, construida con Ruby y Sinatra. La interfaz está pensada para ser clara, rápida y fácil de usar tanto en móvil como en escritorio.

## Qué incluye

- Tablero 4x4 con movimientos y fusiones de fichas
- Controles por teclado y gestos en pantalla
- Puntuación y mejor puntuación
- Tema claro y oscuro
- Sonido opcional
- Guardado automático del progreso

## Requisitos

- Ruby 3.1 o superior
- Bundler

## Inicio rápido

```bash
bundle install
ruby app.rb
```

Abre http://127.0.0.1:4567 en tu navegador.

## Notas importantes

- El progreso se guarda automáticamente en la carpeta data.
- El juego está pensado para usarse de forma local y sencilla.
